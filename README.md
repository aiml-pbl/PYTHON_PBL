The Repository features three folders: PHASE 1, PHASE 2, PHASE 3.

PHASE 1 --> Features Raw Dataset, EDA, Preprocessing, and the data cleaning methods.
Result of PHASE 1 --> Final Cleaned dataset to be used for model training.

PHASE 2 --> Features training multiple models on Regression, all the models were explicitly trained under 82:20 ratio of data split and evalution for all of them.
Result of PHASE 2 --> Linear Regression with Log Transformation emerged as the best model to be trained.

PHASE 3 --> Features the best selected model, in this we trained this model under three data splits: 80:20, 70:30, 60:40 and evaluated the best split, 
trained the final model and added a way for the user to enter the values to be predicted under the final model.
Result of PHASE 3 --> The final trained model and user input values with final prediction.
